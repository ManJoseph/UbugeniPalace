# UbugeniPalace - Documentation Summary

## Project Overview
**UbugeniPalace** is a comprehensive e-commerce platform designed to showcase and sell authentic Rwandan handmade crafts, connecting local artisans with customers worldwide. The project addresses real community needs while demonstrating full-stack web development skills.

## Key Project Highlights

### 🎯 **Real-World Problem Solved**
- **Problem:** Rwandan artisans struggle to reach customers beyond their immediate community
- **Solution:** Digital marketplace providing global market access
- **Impact:** Economic empowerment and cultural preservation

### 🛠️ **Technologies Used**
- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **Backend:** PHP 8.0+, MySQL 8.0+
- **Server:** Apache (XAMPP/WAMP)
- **Security:** Password hashing, SQL injection prevention, XSS protection

### 👥 **User Roles Implemented**
1. **Customers** - Browse, purchase, review products
2. **Artisans** - Upload products, manage inventory, track sales
3. **Admins** - System management, user approval, analytics

### 🏗️ **System Architecture**
```
Frontend (HTML/CSS/JS) → Backend (PHP) → Database (MySQL)
```

## Database Design

### **9 Main Tables**
1. **users** - User accounts and authentication
2. **artisans** - Artisan-specific profiles
3. **categories** - Product categorization
4. **products** - Product listings and details
5. **orders** - Order management
6. **order_items** - Order line items
7. **cart** - Shopping cart functionality
8. **reviews** - Product reviews and ratings
9. **wishlist** - User wishlist management

### **Key Relationships**
- Users ↔ Artisans (One-to-One)
- Artisans → Products (One-to-Many)
- Categories → Products (One-to-Many)
- Users → Orders (One-to-Many)
- Products → Reviews (One-to-Many)
